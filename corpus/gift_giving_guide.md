## Ordering as a Gift

### Information You'll Need

**About the child:**
- First name
- Age or birthdate
- Photo (request from parents)
- Preferred pronouns

**Optional information:**
- Specific challenges or fears
- Family members to include
- Pets
- Interests

### Getting Photos from Parents

**How to request:**

**Email template:**
```
Hi [Parent's name],

I'd love to give [Child's name] a personalized 
storybook for [occasion]. Could you send me a 
recent photo of [him/her/them]? 

The photo should be:
- Clear and well-lit
- Front-facing
- At least 1200x1600 pixels

I'll keep it confidential and only use it for 
the book. Thanks!
```

**Text message template:**
```
Hi! I want to get [Child] a personalized book 
for [occasion]. Can you send a good photo?
```

## Group Gifts

### Organizing Group Gifts

**Perfect for:**
- Classroom gifts
- Office collections
- Family group gifts
- Friend groups

**How to organize:**

1. **Collect contributions:**
   - Set target amount
   - Use Venmo, PayPal, etc.
   - Track contributors

2. **Decide on book:**
   - Poll for theme preference
   - Get photo from parents
   - Coordinate customization

3. **Order together:**
   - One person places order
   - Include all names in dedication
   - Split shipping cost

**Dedication example:**
```
To Emma,

Congratulations on starting kindergarten! 
We're all so proud of you!

Love from,
Mrs. Johnson's Preschool Class
[List of names]
```

## Gift Etiquette

### Asking for Information

**Polite ways to get details:**

**For photo:**
- "I'd love to give [Child] a personalized book. Could you send a photo?"
- Be specific about photo requirements
- Assure privacy

**For preferences:**
- "What challenges is [Child] facing right now?"
- "Any upcoming milestones?"
- "What would be most helpful?"

### Respecting Privacy

**Important:**
- Don't share child's photo
- Don't post about gift before giving
- Respect parent's preferences
- Ask before including siblings

### Gift Receipts

**We provide:**
- Gift receipt (no prices)
- Return/exchange information
- Contact details

**Parents can:**
- Exchange for different theme
- Upgrade format
- Modify customization

## Gift FAQs

**Q: Can I order without a photo?**
A: Yes! Purchase gift card and parents can provide photo.

**Q: What if I don't know the child's age?**
A: Estimate is fine. We adjust content appropriately.

**Q: Can I see the book before giving?**
A: Yes! Ship to yourself first, or request digital preview.

**Q: What if parents already ordered the same theme?**
A: We can exchange for different theme or format.

**Q: Can I include multiple gift-givers in dedication?**
A: Yes! List all names in custom dedication.

**Q: Is gift wrapping available?**
A: Yes, $5 per book. Select at checkout.
